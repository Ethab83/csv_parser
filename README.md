# csv_parser

csv parser to json

> [!NOTE]
> compatible with **python 3.10.12+**

## Install dependencies

```shell
pip install -r requirements.txt
```

## Usage

```shell
python main.py file.csv [-t | -l]

    -t: keys are located on top row (default)
    -l: keys are located on left column

# show help
python main.py -h
```
